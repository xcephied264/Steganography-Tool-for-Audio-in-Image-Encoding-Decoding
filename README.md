# Steganography-Tool-for-Audio-in-Image-Encoding-Decoding
Developed an image-based steganography system to encode and decode audio data within images
Implemented algorithms to convert audio signals into embeddable binary data and integrate them into image pixel structures
Designed decoding mechanisms to reconstruct audio data from modified images
Addressed challenges related to data size, signal integrity, and extraction accuracy
Applied concepts of covert communication, data hiding, and multimedia security
Implemented LSB-based steganography by embedding binary-encoded data into the least significant bits of audio samples
